# Employee Management System

## About
Employee Management System is a Python-based Object-Oriented Programming project designed to manage Person, Employee, and Manager records through an interactive menu-driven interface.

This project demonstrates core OOP concepts such as classes, inheritance, multiple inheritance, encapsulation-like structure, and data handling while improving programming logic and software development skills.

## Features
- Create Person records
- Create Employee records
- Create Manager records
- Display stored information
- Input validation for better accuracy
- Interactive menu-driven system

## OOP Concepts Used
- Classes & Objects
- Inheritance
- Multiple Inheritance
- Function Definitions
- Method Calling
- Data Management
- Input Validation

## Technologies Used
- Python

## Purpose
The main purpose of this project is to strengthen understanding of Object-Oriented Programming concepts and improve logical thinking through practical implementation.
